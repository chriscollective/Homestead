# Homestead Planner 祖傳家園規劃系統

以《鳴響雪松》祖傳家園理念為核心的土地規劃工具:在數位環境中設計自己的一公頃家園,繪製真實的不規則地界、配置森林/果樹/菜園/水體,並以祖傳家園設計原則(森林覆蓋 ~50%、水源配置)做即時檢查。

完整規格與路線圖見 [`docs/homestead-planner-spec.md`](docs/homestead-planner-spec.md)。

## 目前進度:Phase 1 平面規劃 MVP ✅

| 模組 | 內容 | 狀態 |
|---|---|---|
| 專案架構 | Vite + React 18 + TypeScript + Zustand | ✅ |
| M1 土地畫布 | 不規則多邊形地界繪製/頂點編輯(拖曳、插入、右鍵刪除)、縮放平移、1/5/10m 網格、面積(㎡/公頃/坪)與周長、測距工具、比例尺、指北針 | ✅ |
| M2 元素系統 | 喬木單株放置(成熟冠幅投影圈)、林地/菜園/草地區塊、池塘多邊形、拖曳移動、喬木間距警告(依成熟冠幅)、元素屬性面板(物種/種植年份/備註)、undo/redo | ✅ |
| M3 植物資料庫 | 30 種台灣物種靜態資料(13 果樹、11 林木、4 綠籬灌木、2 竹類),含成長曲線、環境需求、氣候分區、產量、功能標籤、原生種標記 | ✅ |
| 設計原則儀表 | 森林覆蓋比例即時儀表(網格取樣,正確處理樹冠與林地重疊)、水源存在提示 | ✅ |
| 本地儲存 | localStorage 自動儲存、匯出/匯入 JSON、匯出 PNG 平面圖 | ✅ |

> 植物成長曲線與產量目前為初版估算值(資料檔內有 `sources` 欄位標註),僅供規劃參考,待後續查證修正。

## 開發

```bash
npm install
npm run dev      # 開發伺服器
npm test         # 單元測試(幾何/成長/指標引擎)
npm run build    # 型別檢查 + 產線建置
```

## 操作說明

- **⬠ 繪製地界**:逐點點擊繪製,點擊起點或雙擊閉合;Esc 取消
- **🌳 放置植栽**:右側面板選物種後點擊畫布放置,圓圈為成熟冠幅投影
- **▦ 繪製區塊**:林地(計入森林比例)/菜園/草地,雙擊完成
- **💧 繪製池塘**:雙擊完成
- **↖ 選取/編輯**:點選元素拖曳移動;選取多邊形後拖曳頂點改形狀、點邊線中點方塊插入頂點、右鍵頂點刪除;Delete 刪除元素
- 滾輪縮放、空白處(或中鍵)拖曳平移;Ctrl+Z / Ctrl+Shift+Z 復原/重做

## 架構原則(摘自規格書 3.3)

- **模擬引擎獨立**:幾何、成長插值、覆蓋率、間距檢查全部是 `src/engine/` 下的純函式,以 Vitest 測試,渲染層不含任何運算邏輯
- **單一資料源**:畫布是 `HomesteadProject` 的純視圖;未來 3D 檢視(Phase 2)共用同一份資料模型
- **本地優先**:編輯即時存 localStorage;Phase 2 再加後端同步
- **座標系統**:內部統一公尺制本地平面座標
